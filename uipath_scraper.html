Automating Challenge

This robot can opens an Excel file, read all cells in the sheet, scrape the data, and fill forms. The Excel sheet can be downloaded from http://www.rpachallenge.com/ and it should be stored inside the projects folder.

Processes

Step 1. Inside the Read Range properties, I edited the Range, the SheetName, the Workbook Path, and created a variable. I set the range to empty ” “ so it can read all cells in the Excel sheet. The Excel Challenge should be stored in the same project folder as the project and the SheetName must have the same name as the sheet inside the Challenge file. I also created a variable called datatable to store the data from the Excel challenge sheet so that I could output that data in other places within our project.
