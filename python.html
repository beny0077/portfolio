Automating with Python

Python with Selenium

Realizing that my job searches was becoming very repetative I sought out to find some way to autmate my job searching task. I found two useful tools to automate my task. One was Selenium to help me interact with a browser and the other was Scrapy, which is useful to download data from the broswer. However, to keep things simple I will just be showing the Selenium framework using Python

Technologies used

  • Python 3.8.5
  • PyCharm Version: 2020.2.2
  • Crome WebDriver 85.0.4183.87

Automated Tasks

  • Visit the website
  • Search a job by its title
  • Select a radius
  • Handle the pop-up window
  • Select a distance for the job
  • Select an experience level

Getting started

Once I had my environment ready I installed Selenium with the following command: 

pip install selenium

Then I installed the Chrome WebDriver for the automation from Chromium and imported the packages into my Python project that I would need for this project: 

from selenium import webdriver
from selenium.webdriver.common.keys import Keys
import time

The imported packages will allow us to use the methods for the program to interact with the browser. Here is list of the methods we will be using:

  • webdriver.Chrome() - Visits the website
  • driver.find_element_by_xpath() - Locates an element
  • driver.find_element_by_id() - Locates an element
  • send_keys() - writes the input
  • click() - Mocks the click of a mouse
  • sleep() - Waits at a specified time before moving to another tasks
  • refresh() - Refreshes the current web page
  • clear() - Clears a text box
  • driver.close() - Closes the automation

Summary

For the most part the whole process was simple but we don't live in a perfect world in I did find the pop-window to be the most challenging part to deal with. I used the Selenium WebDriver Switch method so the Webdriver would move to the pop-up window but nothing seemed to work. Since I wanted this project to be simple, I just use the refresh() method to handle the pop-up. Overall, I am happy with my Python project using Selenium and to be honest this is one of my favorite projects because it makes my life easier. In another project I will be using Scrapy to extract the data from the website and place it into an Excel or Google sheet.

Python Code


from selenium import webdriver
from selenium.webdriver.common.keys import Keys
import time

driver = webdriver.Chrome(executable_path = "C:\Drivers\chromedriver_win32\chromedriver")

# open the website and wait one second before moving on to the next step
driver.get('https://indeed.com/')
time.sleep(1)

# Look for the text box "What" text box to input job title and enter the title
jobTitleSearchBox = driver.find_element_by_xpath('//*[@id="text-input-what"]')
# enter job title
jobTitleSearchBox.send_keys('software')

# Now click on the "Find Jobs" tab and wait a couple of second so things load up nicely before our next step
searchMyJobTitle = driver.find_element_by_xpath('//*[@id="whatWhereFormId"]/div[3]/button')
searchMyJobTitle.click()
time.sleep(2)

# ***************************************************************
# locate the dropdown element for distance and click it to display the options, wait a second before the next step
dropdownRadiusList = driver.find_element_by_id('filter-distance')
dropdownRadiusList.click()
time.sleep(1)

# Now locate the "within 50 miles" option by it's xpath, then wait 2 seconds before the next step
selectFiftyMileRadius = driver.find_element_by_xpath('//*[@id="filter-distance-menu"]/li[6]/a')
selectFiftyMileRadius.click()
time.sleep(2)

# at this point there will be a pop-up menu to to register your email, to deal with this, just refresh the page
driver.refresh()
# ***************************************************************

# Also type in the location
jobLocation = driver.find_element_by_xpath('//*[@id="where"]')
jobLocation.clear()
jobLocation.send_keys('Raleigh, NC')
time.sleep(2)

# Now click on the "Find Jobs" tab and wait a couple of second so things load up nicely before our next step
ButtonSearchMyLocation = driver.find_element_by_xpath('//*[@id="fj"]')
ButtonSearchMyLocation.click()
time.sleep(2)

# Locate the seniority tab and click it to expand the options
seniorityDropDown = driver.find_element_by_id('filter-experience-level')
seniorityDropDown.click()
time.sleep(2)

# Select the entry level option
GiveMeEntryLevel = driver.find_element_by_xpath('//*[@id="filter-experience-level-menu"]/li[2]')
GiveMeEntryLevel.click()
time.sleep(15)

driver.close()
